# MiniHunt

Developed with Unreal Engine 5

游戏名称：迷你猎杀

故事背景：
在未来，世界范围内突然出现大量魔兽，他们占据了许多人类的城市，人们被迫离开家乡建设新的家园。
在乱世之中有一些人开辟了一条新的生存道路，去魔兽的据点寻找争夺资源。
这些人被世间称为“猎人”

游戏玩法：
游戏采用局域网联机，一名玩家创建房间，其他玩家可以加入房间，最多支持4人同台竞技。
玩家选择角色并准备，所有人准备好后，房主点击开始游戏进入对局。
每个角[CombatComponent.h](Source/MiniHunt/Character/CombatComponent.h)色有自己的一个主动技能和一个被动技能。

局内每个人出生点不同，出生点有运送货物的提交点（提交积分），玩家需要在地图中搜索物资并提交到货物提交点。
地图中存在武器，弹药等装备。玩家可以拾取装备射杀怪物获取积分点，也可以寻找并拾取物品获得积分点，身上的积分点只有提交后才计分。
玩家也可以选择抢夺其他玩家的积分点。玩家若被其他玩家击杀，则被击杀玩家的积分点损失一半给与抢夺者。
如果被魔兽击杀，同样损失一半积分点。
玩家死亡后会复活在出生点。
10分钟游戏结束，积分多者获胜。
